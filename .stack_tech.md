# Stack Tecnológica

## Frontend
- **Framework:** React 19
- **Build Tool:** Vite 8
- **Roteamento:** React Router DOM
- **UI/Estilização:** Radix UI Themes, Lucide React (Ícones)
- **Componentes Avançados:** 
  - Drag and Drop: `@dnd-kit/core`, `@dnd-kit/sortable`, `@dnd-kit/utilities`
  - Gráficos: `recharts`
  - Gráfico de Gantt: `gantt-task-react`
- **PWA:** `vite-plugin-pwa`

## Backend & Serviços
- **BaaS:** Firebase 12

## Ferramentas de Qualidade e Scripts
- **Linter:** oxlint
- **Utilitários:** `axios`, `cheerio` (Web Scraping para buscar feriados), `slugify`

## Arquitetura e Padrões
- **Padrão Arquitetural:** Single Page Application (SPA) modularizada baseada em Componentes Funcionais do React (Hooks).
- **Gerenciamento de Estado:** Context API ou estados locais com Hooks.
